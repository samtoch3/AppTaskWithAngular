# AppTaskWithAngular

Petite application de gestions des taches réalisée avec Angular par Samson N.

####  - installer angular
``npm i -g @angular/cli
  ng new taskApp``

####  - lancer le serveur (http://127.0.0.1:4200)
``ng serve``

####  - réalisation des tests
``ng test``

####  - mise en production
``ng build``
